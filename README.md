# Standing

**Локальная репутация как продукт.** Платформа, где зритель донатит стримеру в USDC на Solana и за
это копит непродаваемую, непередаваемую репутацию внутри комьюнити конкретного стримера. Поверх
репутации позже встают мини-игры; в этой версии — только ядро.

> «Standing» — кодовое имя-заглушка (англ. *standing* = и «положение/статус в сообществе», и «турнирная
> таблица»). Замени на финальный нейминг, когда он будет.

## Зачем это

Стримеры зарабатывают на донатах, но донат — разовый акт без следа. Здесь каждый донат превращается
в **накапливаемый статус** внутри канала: тиры, бейджи, место в лидерборде, перки. Зритель получает
видимое, заработанное положение в комьюнити; стример — механику удержания и монетизации. Это
фундамент данных, на котором потом живут игры (задание-донат с проверкой комьюнити и т.д.).

## Ключевые свойства

- **Solana + USDC** — транзакция стоит центы, иначе экономика порогов не работает.
- **Некастодиально** — оператор не держит деньги доната. Донор → стример напрямую.
- **Репутация локальна, детерминирована, перевычислима** — её может пересчитать кто угодно.
- **Текст приватен до показа** — главный фильтр от казусов в прямом эфире — человек в цикле.
- **Управляемо оператором, не цензуроустойчиво** — это требование комплаенса, а не баг.

## Разработка

**Требования:** Node.js 20+ и npm (в окружении, где писался скелет, их не было — версии пакетов в
`package.json` сверь при первом `npm install`).

```bash
npm install        # установить зависимости
npm run dev        # дев-сервер → http://localhost:3000
npm run lint       # ESLint
npm run typecheck  # tsc --noEmit
npm run format     # Prettier
```

Источник данных переключается флагом `NEXT_PUBLIC_DATA_SOURCE` (`mock` | `api` | `chain`, см.
`CLAUDE.md` §3 и `.env.example`). На Фазе 0/1 — `mock`.

**Что посмотреть в Фазе 0:**

- `/` и остальные маршруты — экраны-заглушки под все экраны `frontend/screens.md` (тема видна, роутинг есть).
- `/dev/kitchen-sink` — все дизайн-токены и примитивы (Button/Input/Textarea/Dialog/Tabs/Tooltip/Toast,
  empty/loading/error) + смоук-проверка `useData()` на пустом `MockDataProvider`.

**Структура кода** (отдельно от спек):

```
src/
├── app/                  маршруты (App Router) + layout, globals.css, providers
├── components/ui/        примитивы поверх Radix + feedback (Skeleton/EmptyState/ErrorState)
├── components/layout/    шапка, сайдбар студии, обёртки страниц
└── lib/
    ├── utils.ts          cn + деньги (toMicro/fromMicro/formatUSDC) + адреса
    └── data/             DataProvider (интерфейс, mock, контекст useData, хуки), типы (data-model)
```
