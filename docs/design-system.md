# Дизайн-система

> Визуальный язык, из которого растут все экраны. **Бренд — Crown** (ADR 0022): чёрный с золотом,
> геральдика двора, английский копирайт. Токены — единственный источник истины в `src/app/globals.css`
> (`:root`, CSS-переменные), проброшены в утилиты Tailwind (`tailwind.config.ts`, `theme.extend`).
> Числа (hex, rem, ms) — стартовые значения; меняй осознанно и централизованно через переменные,
> **не** сырыми hex в компонентах. Лексикон бренда — `docs/glossary.md` и `src/lib/crown.ts`.

---

## 1. Направление (тезис дизайна)

Продукт живёт **рядом с прямым эфиром**: донор в моменте, стример держит панель на втором мониторе во
время трансляции, деньги реальные и необратимые. Отсюда тезис — **«пульт трансляции встречает
расчётную книгу»**. Два регистра в одном продукте, и их нельзя путать:

- **Статусный регистр** (Reign, ранги, бейджи, лидерборд) — заработанный, живой, «отчеканенный».
- **Денежный регистр** (Crown, разбивка комиссии 97/3, финальность) — точный, спокойный,
  доверительный. Как квитанция о расчёте. Числа моноширинные, без хайпа.

**Сигнатура Crown** (одна запоминающаяся вещь, на неё тратим всю смелость): **геральдика двора на
чёрном золоте**. Локальная репутация показывается как **ранг** (Squire → Knight → Baron → Duke → King)
с крест-бейджем-монограммой; сезонный топ-1 — **The Crown**, отмеченный **фирменным знаком** `CrownLogo`
(`src/components/crown-logo.tsx`, `currentColor` — красится в money-золото), НЕ ОС-эмодзи 👑 (рендерится
по-разному на платформах, вне бренда). Корона мягко «дышит» золотым свечением (`animate-crown`).
Купить-продать это визуально нельзя — вычисленный титул, а не токен. Всё остальное
вокруг — тихое и строгое, на тёмно-сером.

**Чего избегаем** (это AI-дефолты): кремовый фон + serif + терракота; near-black + один кислотный
акцент; «газетная» вёрстка с волосяными линейками. У нас dark-first, **нейтральный тёмно-серый + два
золота с разными ролями** (Reign vs Crown) + пергаментный текст. Дуальность акцента — структурный
выбор (инвариант §4.7 «деньги ≠ статус»), а не украшение.

## 2. Цвет

Dark-first (контекст OBS / второй монитор). Фон — **нейтральный тёмно-серый** (решение владельца
2026-07-07: чистый чёрный был слишком жёстким); золото живёт в тексте, акцентах и рамках, не в фоне.
Лестница поверхностей сохраняется: bg < surface < surface-2. Светлая тема — позже, на тех же
семантических переменных.

```css
:root {
  color-scheme: dark;

  /* Поверхности — тёмно-серый с нейтральными панелями (без тёплого подтона) */
  --bg:            #15171a;   /* база — тёмно-серый с холодным подтоном */
  --surface:       #1e2024;   /* карточки/панели */
  --surface-2:     #292b30;   /* оверлеи/дорожки/наведение */
  --border:        #8f7536;   /* рамки — приглушённое золото */
  --border-strong: #c9a24a;   /* акцент/hover — яркое золото */

  /* Текст — пергамент (тёплый королевский белый) */
  --text:          #f3ecdd;
  --text-muted:    #ada491;
  --text-faint:    #726a5a;

  /* Акцент СТАТУСА (Reign / ранги) — античное золото, приглушено под money */
  --status:        #c9a24a;
  --status-dim:    #8f7536;
  --status-bg:     #2a2517;

  /* Акцент ДЕНЕГ (Crown / транзакции / финальность) — яркое золото, рационировано */
  --money:         #e4b34c;
  --money-dim:     #b5832f;
  --money-bg:      #2c2616;
  --money-bright:  #f6d778;   /* блик короны / hover (только через var(), не как утилита) */

  /* Семантика (отдельно от акцента) */
  --danger:        #cf5a4d;   --danger-bg: #332120;
  --warn:          #e0a94b;
  --info:          #6e8bff;   /* фокус-кольцо + нейтральные интерактивные акценты */
}
```

**Дисциплина акцентов (правило, не вкусовщина):**
- `--money` (яркое золото) — **только** для подтверждённых/финальных денежных состояний: «Crown прошёл»,
  финальная сумма. Никогда как декор. Его редкость = его смысл.
- `--status` (античное золото, чуть темнее money) — ранги, бейджи, прогресс Reign, лидерборд.
- `--info` (синий) — фокус-кольцо и обычные интерактивные акценты (ссылки, активные табы), чтобы не
  «жечь» золото двумя смыслами.
- `--danger` — held/blocked/деструктив/ошибка.

Ранги (Crown) берут дефолтные цвета-«металлы» из `src/lib/crown.ts` — прогрессия камень → сталь →
бронза → аметист → золото:
`Squire #8a8577 · Knight #bac1c8 · Baron #b07a46 · Duke #9a79c4 · King #c9a24a`.
Стример может назвать свои тиры иначе; для **дисплея** ранг на личных экранах всегда считается по очкам
Reign (лестница `RANKS`), независимо от названий тиров в конфиге.

**Утилиты Tailwind** (`tailwind.config.ts`): цвета — `bg`, `surface`/`surface-raised`,
`border`/`border-strong`, `fg`/`fg-muted`/`fg-faint` (текст), `status`/`status-dim`/`status-bg`,
`money`/`money-dim`/`money-bg`, `danger`/`danger-bg`, `warn`, `info`. Компоненты ссылаются на них,
не на сырые hex.

## 3. Типографика

**Один UI-шрифт — Inter** (и заголовки, и корпус). Моноширинный — **JetBrains Mono** — строго для
чисел/сумм/адресов/tx (структурное правило: числа = данные). Без декоративных серифов. Оба через
`next/font/google` с `display: swap`, сабсеты latin+cyrillic.

```css
:root {
  --font-body: Inter;            /* заголовки и корпус — один шрифт */
  --font-mono: "JetBrains Mono"; /* числа/адреса/суммы/tx (tabular-nums) */
  /* --font-display в tailwind оставлен для совместимости, но не задаётся: display-роли идут body-шрифтом */
}
```

Шкала (type scale, 1.250 — major third), display-роли — Inter с плотным трекингом (классы
`.text-display-xl` … `.text-caption` в `globals.css`):

| Роль | Размер | Вес | Шрифт | Использование |
|------|--------|-----|-------|---------------|
| Display XL | 3.05rem | 600 | Inter | hero / rank-up |
| Display L | 2.44rem | 600 | Inter | заголовки экранов (`Personal Space`, `Settings`) |
| H1 | 1.95rem | 600 | Inter | секции |
| H2 | 1.56rem | 600 | Inter | подсекции |
| H3 | 1.25rem | 600 | Inter | карточки |
| Body | 1.0rem | 400 | Inter | основной текст |
| Small | 0.875rem | 400 | Inter | мета |
| Caption | 0.75rem | 500 | Inter | лейблы, eyebrow (uppercase, tracking +0.06em) |
| **Mono / Amount** | по контексту | 500 | **JetBrains Mono** | **все суммы, адреса, очки Reign, tx** |

Крупные display-роли ужимаются на `≤640px` (мобильная адаптация в `globals.css`). Суммы — `tabular-nums`
(`.mono`), чтобы цифры не «прыгали». Адреса усечённо `7xKp…3fQa`, по клику — копирование.

## 4. Пространство, ритм, форма

```css
:root {
  --space-1: 4px;  --space-2: 8px;  --space-3: 12px; --space-4: 16px;
  --space-5: 24px; --space-6: 32px; --space-7: 48px; --space-8: 64px;

  --header-h:  60px;   /* высота липкой шапки — один источник для offset липких сайдбаров (rail-pinned-*) */
  --radius-sm: 6px;   --radius: 10px;  --radius-lg: 14px;  --radius-pill: 999px;
  --maxw:      1200px; /* контентный контейнер (утилита max-w-content) */
}
```

- Контент-страницы — одна колонка `--maxw`; дашборды — сетка с желобом `--space-5`.
- Карточки — `surface`, бордер `border` (золото), радиус `--radius`. Приподнятые (модалки/поповеры/
  наведение) — `surface-raised`.
- Тени минимальны (dark-тема): глубину даём бордером-золотом и фоном, не размытой тенью. Тонкая тень —
  только под модалками/оверлеем и на hover карточек realm (золотое свечение).
- Липкие панели — `.rail-pinned-left` / `.rail-pinned-right` (top = `--header-h + --space-4`), в потоке
  грида, без fixed/JS.
- **Крест-бейдж ранга** (сигнатура): не плоский чип, а геральдический знак — монограмма-сигил в
  «металле» ранга. Единственное место, где позволяем «дорогую» деталь.

## 5. Движение

Сдержанно и осмысленно. Лишняя анимация = ощущение «сгенерировано AI». Все токены и кейфреймы — в
`globals.css`; всё уважает `prefers-reduced-motion` (глобальный гаситель в конце файла).

```css
:root {
  --ease: cubic-bezier(0.2, 0.7, 0.2, 1);
  --dur-fast: 120ms; --dur: 200ms; --dur-slow: 320ms;
}
```

- **Бюджет смелости — оркестрованные моменты**, остальное мгновенно и тихо:
  1. **Финальность Crown** — две суммы расходятся 97/3, затем «печать» (`animate-stamp`), сумма
     переключается в `--money`. «Деньги ушли, необратимо».
  2. **Rank-up** — знак ранга прочеканивается один раз при переходе.
  3. **The Crown** — корона топ-1 в Realm roll мягко «дышит» золотым свечением (`animate-crown`,
     низкая амплитуда, медленный цикл — не отвлекает).
- **Переход между страницами** — нативный View Transitions API (кроссфейд снимков; триггерит
  `PageTransitions` при клиентской навигации, длительность `--dur-slow`). При старте сайта — мягкое
  проявление (`animate-enter`, только opacity — безопасно для sticky/fixed).
- **Появление сеток/списков** — каскад `enter-stagger` (fade + лёгкий подъём со сдвигом); смена
  сортировки/фильтра — симметричный `animate-list-out` → подмена → каскад-вход.
- **Меню/дропдауны** — зеркальные `menu-in`/`menu-out` (delayed unmount на `onAnimationEnd`).
- **Скелетоны** — золотистый блик слева направо (`skeleton-shimmer`), в тон металлу, а не пульсация.
- Hover/focus — `--dur-fast`, только бордер/фон, без «прыжков». Фокус-кольцо — `--info`.

## 6. Качество-флор (без объявлений)

- Адаптив до мобильного (донор часто с телефона; дашборд/оверлей — десктоп).
- Видимый фокус с клавиатуры (`:focus-visible` → 2px `--info`) на всех интерактивных элементах.
- Контраст ≥ WCAG AA для текста; проверять `--text-muted` на `--surface`/`--bg`.
- Empty/loading/error/disabled — спроектированы, не «дыры». **Никаких заглушек-обманок** (CLAUDE.md §7):
  пустой мир в `mock` показывает честные empty-states, а не фейковые realms.

## 7. Копирайт (тон интерфейса)

**UI — английский** (бренд Crown, ADR 0022). От лица пользователя, активный залог, sentence case,
без филлера.

- Кнопка говорит, что произойдёт: «Crown» (задонатить), «Show», «Hide», «Open a realm»,
  «Ban wallet». Действие сохраняет имя через весь флоу (кнопка «Show» → тост «Shown»).
- **Финальность — честно:** перед подписью — «Crowns are final. No refunds.» После — «Done. The
  streamer got it.» Никаких «maybe/usually».
- Лексикон бренда (везде одинаково): donation → **Crown**, reputation → **Reign**, tier → **rank**,
  channel → **realm**, сезонный топ-1 → **The Crown**. Полная карта — `docs/glossary.md`.
- **Никаких обещаний справедливости** (`docs/legal-and-risk.md` §6): рамим как статус/комьюнити, не
  «выигрыш/гарантия».
- Ошибки не извиняются и не туманят: что случилось + как починить. Пустой экран — приглашение к действию.
- Адреса/суммы — данные, не проза: моно, копируемы.

## 8. Реализация

- Токены — в `:root` (`src/app/globals.css`) + проброс в `tailwind.config.ts` через `theme.extend`
  (цвета, spacing, fontFamily, radius, timing). Компоненты ссылаются на семантические утилиты/переменные
  (`text-fg`, `bg-surface`, `text-money`, `border-border`), **не** на сырые hex. Светлая тема позже =
  переопределение переменных, без правки компонентов.
- Шрифты — `next/font/google` (Inter + JetBrains Mono) в `src/app/layout.tsx`, `display: swap`, проброс
  через `--font-body` / `--font-mono`.
- Бренд-слой (ранги/лексикон/относительное время) — `src/lib/crown.ts`; логотип — `src/components/crown-logo.tsx`.
- `/dev/kitchen-sink` — витрина всех токенов и компонентов для визуального ревью.
